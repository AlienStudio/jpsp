User Table

Name Data Type Notes
id bigint index
username text
password longtext MD5
grade int
classroom int
jie int 第几届
club text 加入的社团
last_five_times_login_IP text 最近五次登陆的IP

Club Table

Name Data Type Notes
id int
name text
proprieter text 社长
proprieter_qq int 社长QQ
if_enroll int 是否招新,只有0和1两种Value
enroll_group_QQ int 招新群QQ
email text
label text 社团标签，比如科技，人文这样的
是否已通过注册 int Value:0 or 1 用来判断是不是创建后需要审批的社团
stars float 社团评优时进行打分
club_introduction text 社团自我介绍

Post Table

Name Data Type Notes
id bigint
post_author text 发布动态的社团或活动组委会
post_time datatime 发布时间
post_status int Value: 0 or 1 ,是否批准发布
post_title text 发布的标题
comment_status int Value: 0 or 1, 是否可以评论
category text 类别：比如是社团动态，社团招新，等等
post_content longtext

Comment Table
Name Data Type Notes
id bigint
comment_post_id int
comment_datetime datetime
comment_content longtext
comment_author id 用id作为标识符
comment_author_IP text

Admin Table

Name Data Type Notes
id int
name text
password longtext MD5
authority int Value:0 or 1 or 2 or 3
1:活动组委会
2:社团部
3:教师
4:管理员
last_five_times_login_IP text
phone_number int
